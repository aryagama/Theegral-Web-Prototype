# Theegral Web Prototype (Next.js)

Migrated from static multi-page HTML/CSS/JS into Next.js App Router while keeping the same design and behavior.

## Routes

- `/` (home)
- `/works`
- `/directory`
- `/about`
- `/work-1`

Legacy URLs are redirected:

- `/index.html`
- `/works.html`
- `/directory.html`
- `/about.html`
- `/work-1.html`

## Run

1. Install Node.js 18.17+ (or Node.js 20+).
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
4. Open `http://localhost:3000`
